## Key Messaging

- **FIRST** open-source security scanner with AI safety content detection
- Detects threats traditional scanners are blind to (jailbreak, abliteration, prompt injection, NSFW pipelines)
- Three-tier reporting: Community (free Telegram alerts), Pro (JSON), Enterprise (PDF + SIEM)
- SKSentry — crowdsourced AI threat intelligence (when one sees a threat, everyone's protected)
- Sovereign override via whitelist — security that respects the operator
- Built by practitioners running AI agents in production (dogfooded on 68+ skills)

## Brand Voice

Confident. Technical but accessible. Slightly irreverent. Not corporate-speak.
Built by engineers, for engineers. Community-powered threat defense energy
applied to the AI security space.
